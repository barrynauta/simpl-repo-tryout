# Canonical Location

**This folder represents an external open-source product, not Simpl-Open code.**

The production code lives upstream (at the OSS project's home — Apache, Eclipse, GitHub, etc.), not in `code.europa.eu/simpl/simpl-open/development`. This catalogue folder documents how Simpl-Open uses or wraps the upstream product.

Status: `external-oss`. If you want to record a specific upstream URL, edit `.canonical.yaml` directly.
