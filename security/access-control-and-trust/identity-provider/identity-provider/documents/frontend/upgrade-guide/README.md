# Upgrade guide

## Upgrade from 2.9.x to 2.10.x

The 2.10.x release contains the **eIDAS Configuration** with the EIDAS_ENABLED property.

This component now manages all the Onboarding Notary functionalities.

This component now manages all the Onboarding Templates Management functionalities.

Refer to the [Onboarding Notary](../user-manual/ONBOARDING-NOTARY.md) and [Onboarding Templates Management](../user-manual/ONBOARDING-PROCEDURE-TEMPLATE.md) documentation for more information.


The upgrade from 2.9.x to 2.10.x follows the instructions in the [Deployment Guide](../deployment-guide/README.md).

## Upgrade from 2.8.x to 2.9.x

The 2.9.x release does not require any configuration changes. The upgrade from 2.8.x to 2.9.x requires no special actions.

## Upgrade from 2.7.x to 2.8.x

The 2.8.x release does not require any configuration changes. The upgrade from 2.7.x to 2.8.x requires no special actions.

## Upgrade from 2.6.x to 2.7.x

To perform the upgrade, uninstall the **simpl-fe** component (version **2.6.0**) and install the new **2.7.0** release, following the instructions in the [Deployment Guide](../deployment-guide/README.md).
