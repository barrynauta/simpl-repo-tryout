# openbao-init

Helm chart for the OpenBao initialisation job. Seeds per-component KV paths so [openbao](../README.md) is ready to serve secrets to the rest of the platform on first boot. Companion artefact to the [openbao](../README.md) chart, not a standalone deployable.

The production code lives at the canonical location — see [CANONICAL.md](CANONICAL.md). Machine-readable form: [`.canonical.yaml`](.canonical.yaml).
