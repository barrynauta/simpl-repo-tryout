# fe-security-attribute-provider

Frontend (Angular) used by the Governance Authority to manage data-space identity attributes and their assignment to participants. UI counterpart of the [Security Attributes Provider](../security-attributes-provider/README.md) backend.

## Purpose

- Manage the identity attributes of the data space (create, edit, deactivate).
- Assign identity attributes to participants.

The production code lives at the canonical location — see [CANONICAL.md](CANONICAL.md). Machine-readable form: [`.canonical.yaml`](.canonical.yaml).
