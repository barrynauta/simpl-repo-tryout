# fe-identity-provider

Frontend (Angular) that lets organisations manage dataspace participants and their credentials inside the Simpl-Open agent. UI counterpart of the [Identity Provider](../identity-provider/README.md) backend microservice.

## Purpose

- Participant management (create, list, update, deactivate).
- Credentials management (issue, renew, revoke, view status).

The production code lives at the canonical location — see [CANONICAL.md](CANONICAL.md). Machine-readable form: [`.canonical.yaml`](.canonical.yaml).
