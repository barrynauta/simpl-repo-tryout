# fe-authentication-provider

Frontend (Angular) that lets organisations manage security settings and credentials for their Simpl-Open agent across both authentication tiers. Sibling to the [Tier 1 Authentication Provider](../tier-1-authentication-provider/README.md) and [Tier 2 Authentication Provider](../tier-2-authentication-provider/README.md) backends.

## Purpose

- Manage security settings of the agent.
- Manage participant credentials — generation and installation.
- Test connectivity to other dataspace participants via Tier 2 communication.

The production code lives at the canonical location — see [CANONICAL.md](CANONICAL.md). Machine-readable form: [`.canonical.yaml`](.canonical.yaml).
