# tier2-catalogue-client

Java library providing Tier 2 (mTLS) Feign clients for talking to catalogue services — packages: `tier2/feign`, `tier2/client/catalogueadapter`, `tier2/client/federatedcatalogue`. Used by participant-side services that need to query the catalogue or federated catalogue over Tier 2.

The production code lives at the canonical location — see [CANONICAL.md](CANONICAL.md). Machine-readable form: [`.canonical.yaml`](.canonical.yaml).
