# Upgrade guide

This is a Keycloak provider component, no upgrade needed. See [installation](../installation-guide/README.md) instructions

## Upgrade to v2.6.x

**Update** the environment variable `USERS_ROLES_BASE_URL` (or `USERS_ROLES_BASE_URL` for multi realm mode) from V1 to Tier 1 V2 baseurl

## Upgrade to v2.5.x

**Update** the environment variable `AUTHENTICATION_PROVIDER_BASE_URL` (or `AUTHENTICATION_PROVIDER_URLS_MAP` for multi realm mode) from V1 to Tier 1 V2 baseurl 

See the [required environment variables](../installation-guide/README.md#required-environment-variables) for more details
