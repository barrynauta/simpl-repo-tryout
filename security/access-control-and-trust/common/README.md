# Common (shared IAA libraries)

Shared libraries and contract artefacts used across the Access Control and Trust services (authentication, authorisation, identity provider, security attributes). Not deployable on its own; consumed by the runtime services in this capability.

## Solutions

- [iaa-common](iaa-common/README.md) — shared Java library module providing reusable components, models, utilities, and the `simpl-api-iaa` submodule that centralises all OpenAPI/AsyncAPI specifications for the IAA stack.
