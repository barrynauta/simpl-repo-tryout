# Search engine

Consumer and provider-side search interface for discovering resources in the Catalogue. Supports quick search and advanced search using schema-defined fields, with policy-filtered query forwarding to the Catalogue.

## Solutions

- [Catalogue Client Application](catalogue-client-application/README.md) — search frontend and backend (Java + Angular) with Validation Backend, Contract Consumption Adapter, and EDC Connector Adapter sub-components.
