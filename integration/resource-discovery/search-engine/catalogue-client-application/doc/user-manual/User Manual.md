# Search Client Backend

## Steps to Test

### 1. Accessing the Catalogue UI
- Open the **Catalogue UI** and log in with the user `consumer.sd`.  
  *(Other users will not work.)*

### 2. Search for a Valid Document
Content search in the catalog can be performed in two ways: **Quick Search** and **Advanced Search**.
For **Quick Search**, the user execute following steps:
1. Selects the 'Quick Search' tab
2. Enters a keyword to direct the search
3. Upon clicking the 'Search' button, a list of suggested results will be displayed.

![Search](../../images/quicksearch_1.png)

For **Advanced Search**, the user execute following steps:
1. Selects the 'Advanced Search' tab and select the type of content to target your search. After making a selection, a list of fields available for the search will be displayed.

![Search](../../images/advsearch_1.png)

2. Complete the fields to refine your search

![Search](../../images/advsearch_2.png)

3. Click the 'Search' button to display the search results.

![Search](../../images/advsearch_3.png)

After completing the search, you can click the **'Request Result'** button to view the details of the selected item

![Search](../../images/requestresult.png)
