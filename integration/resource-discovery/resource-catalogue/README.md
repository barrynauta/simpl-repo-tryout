# Resource catalogue

Central publication point for signed self-descriptions in the Simpl-Open data space. Provides secure API for publishing, querying, and managing self-descriptions with syntax, semantic, and quality-rule validation.

## Solutions

- [Simpl Catalogue](simpl-catalogue/README.md) — XFSC Federated Catalogue fork implementing the Governance Authority's central self-description registry with Search Engine, Vocabulary Datastore, and Validation Services.
