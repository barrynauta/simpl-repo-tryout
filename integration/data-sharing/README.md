# Data sharing

> **Status: Not yet implemented.** No business service or solution under this capability has been realised in the current catalogue. The planned services below are sourced from [foundations/capability-map.md](../../foundations/capability-map.md) and represent the intended scope.

Allows participants to exchange data with others through interoperable interfaces.

## Planned business services

- **Bulk data transfer** — moves large datasets reliably with checkpointing, integrity checks, and resume support.
- **Data streaming** — publishes and subscribes to real-time event flows with ordering, retention, and replay.
- **Simple data transfer** — provides lightweight pull or push exchanges for small files and APIs.
