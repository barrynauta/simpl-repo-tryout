# Integration

Integration services connecting data space participants for resource discovery, catalogue publication, and resource sharing.

## Capabilities

- [Application sharing](application-sharing/README.md) — interoperable interfaces for sharing applications, services, algorithms, and ML models. _(Not yet implemented)_
- [Contract enforcement](contract-enforcement/README.md) — programmatic validation and enforcement of contractual terms across interactions. _(Not yet implemented)_
- [Data sharing](data-sharing/README.md) — bulk transfer, streaming, and simple data transfer interfaces between participants. Bulk data transfer is implemented today inside the Connector's S3 Object Storage Extension; simple data transfer (eDelivery) and data streaming are roadmap items.
- [Federated management](federated-management/README.md) — identity, catalogue, and trust federation across data spaces. _(Not yet implemented)_
- [Policy enforcement](policy-enforcement/README.md) — runtime enforcement of access and usage policies at integration interfaces. _(Not yet implemented)_
- [Resource discovery](resource-discovery/README.md) — catalogue publication, search, and metadata validation for data space resources.
- [Resource sharing](resource-sharing/README.md) — data space protocol-based contract negotiation and data exchange.
- [Supporting integration services](supporting-integration-services/README.md) — persistent resource address management across federated environments. _(Not yet implemented)_
