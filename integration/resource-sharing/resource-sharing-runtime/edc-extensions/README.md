# edc-extensions

Simpl-built extensions for the Eclipse Dataspace Connector — packages additional connector capabilities (e.g. S3 object storage, triggering hooks, IAA glue) that plug into the [Gaia-X Connector](../gaia-x-connector/README.md) and the deployment variants ([authority-gaiax-edc](../authority-gaiax-edc/README.md), [consumer-gaiax-edc](../consumer-gaiax-edc/README.md)).

The production code lives at the canonical location — see [CANONICAL.md](CANONICAL.md). Machine-readable form: [`.canonical.yaml`](.canonical.yaml).
