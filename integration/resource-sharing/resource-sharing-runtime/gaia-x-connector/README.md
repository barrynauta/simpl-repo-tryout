# gaia-x-connector

Gaia-X EDC Connector — the upstream Eclipse Dataspace Connector (EDC) fork maintained by the Gaia-X / Simpl team. Used by the [Connector](../connector/README.md) solution as the runtime base for DSP-based contract negotiation and data-plane transfer.

The production code lives at the canonical location — see [CANONICAL.md](CANONICAL.md). Machine-readable form: [`.canonical.yaml`](.canonical.yaml).

> Folder name `gaia-x-connector` (descriptive); upstream gitlab repo is `gaia-x-edc/Connector`.
