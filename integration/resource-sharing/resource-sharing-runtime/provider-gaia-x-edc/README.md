# provider-gaia-x-edc

Provider-side deployment variant of the Gaia-X EDC connector — packaging tailored for the data/application/infrastructure provider participant node. Sibling of [authority-gaiax-edc](../authority-gaiax-edc/README.md) and [consumer-gaiax-edc](../consumer-gaiax-edc/README.md); built on the [Gaia-X Connector](../gaia-x-connector/README.md) base with Simpl-built [EDC Extensions](../edc-extensions/README.md) plugged in.

The production code lives at the canonical location — see [CANONICAL.md](CANONICAL.md). Machine-readable form: [`.canonical.yaml`](.canonical.yaml).
