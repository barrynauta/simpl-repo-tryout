# connector-model-common

Shared Java models for the EDC dataspace connector — `configuration`, `contract`, `resourceaddress`, `catalog`, `transfer` packages. Used by every service that exchanges connector-shaped payloads with the EDC.

Consumed across participant-side services (provider, consumer) that wrap or integrate with the EDC connector.

The production code lives at the canonical location — see [CANONICAL.md](CANONICAL.md). Machine-readable form: [`.canonical.yaml`](.canonical.yaml).
