# Vocabulary Management

Manages ontologies and controlled vocabularies used across the Simpl-Open data space to ensure semantic consistency in self-descriptions and resource discovery. This is a planned component; the architecture specification contains a heading but no body description for this service.

Capability-map placement: `data / semantics-and-vocabulary / vocabulary-hub / vocabulary-management`. This solution implements part of the **Vocabulary hub** business service.
