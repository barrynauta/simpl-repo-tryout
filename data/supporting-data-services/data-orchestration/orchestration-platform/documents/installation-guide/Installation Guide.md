# Installation Guide

## Overview
The local installation guide of **dagster**, along with all required configuration files (including Docker and docker-compose assets), is maintained in the following repository → [Dagster Dev Local](https://code.europa.eu/simpl/simpl-open/development/orchestration-platform/dagster-dev-local).

Local deployment, executed via Docker and docker-compose, is provided exclusively for development purposes and must not be used for production scenarios.
