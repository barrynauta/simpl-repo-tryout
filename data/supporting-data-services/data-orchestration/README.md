# Data orchestration

Execution and management of data workflows, enabling providers and consumers to process, preprocess, and transform data using custom or built-in data services (e.g., anonymisation).

- [Orchestration Platform](orchestration-platform/README.md) — Dagster-based workflow orchestration engine with CI/CD, version control, and Simpl-Open catalogue integration.
