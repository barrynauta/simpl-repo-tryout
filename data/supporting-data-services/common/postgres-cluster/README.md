# postgres-cluster

Helm chart deploying the shared PostgreSQL cluster used as the pervasive persistence layer across Simpl-Open (Keycloak, EJBCA, identity/onboarding/user-roles, catalogue metadata, infrastructure-provider storage). Bundled by [Common Components](../../../../cross-cutting/agents/common-components/README.md) into every agent deployment.

The production code lives at the canonical location — see [CANONICAL.md](CANONICAL.md). Machine-readable form: [`.canonical.yaml`](.canonical.yaml).
