# Field-Level Pseudo-Anonymisation - Installation Guide

## Overview

The **Field-Level Pseudo-Anonymisation** repository is a Dagster code location that provides privacy-preserving transformations for both structured (CSV/DataFrame) and unstructured (text) data.

The complete kubernetes-based deployment guide for production environments is maintained in the following repository → [Dagster](https://code.europa.eu/simpl/simpl-open/development/orchestration-platform/dagster).

The complete local installation guide for running these pipelines, along with all required configuration files (including Docker and docker-compose assets), is maintained in the following repository → [Dagster Dev Local](https://code.europa.eu/simpl/simpl-open/development/orchestration-platform/dagster-dev-local).

Local deployment, executed via Docker and docker-compose, is provided exclusively for development purposes and must not be used for production scenarios.
