## Overview

None.

## 3C.1 - Access Control - End-Users Role Request

Description Simpl shall allow end users, both local and federated, to submit role requests to enable them to operate within the Agent. Simpl shall, at a minimum, enable end users to perform the following actions: Facilitate the role request submission. Facilitate the role request review.
