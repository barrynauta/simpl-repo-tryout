# Non-Functional Requirements

Simpl-Open non-functional requirements, documented from the public requirements catalogue at https://simpl-programme.ec.europa.eu/book-page/simpl-requirements.

Each file contains a summary in the author's own words, a canonical source link, a "Measurable thresholds" section (quoting verbatim where available), and auto-inferred cross-references to the capability map (verify before relying on them).
