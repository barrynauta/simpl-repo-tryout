# Canonical Location

The Archi / ArchiMate models for Simpl-Open live at:

<https://code.europa.eu/simpl/simpl-open/architecture/-/tree/master/model?ref_type=heads>

The repository as a whole (containing the model, the FTA specification document, and this coArchi tooling guide) is at:

<https://code.europa.eu/simpl/simpl-open/architecture>

The models are edited collaboratively via the Archi **coArchi** plugin pointing at `git@code.europa.eu:simpl/simpl-open/architecture.git` on the **master** branch — see [README.md](README.md) for the full procedure.

This folder in the catalogue is a structural pointer. The authoritative source files (`*.archimate`, model fragments under `model/business/`, `model/technology/`, `model/relations/`, `model/diagrams/`, `model/other/`) are not mirrored here; clone the upstream repository via coArchi to view or edit them.
