# User roles

Application-layer user and role management on top of the Tier 1 Authentication Provider (Keycloak). Allows Governance Authority and participant administrators to create users, define roles, and assign identity attributes.

## Solutions

- [fe-users-roles](fe-users-roles/README.md) — Angular frontend for the User-roles business service. The backend lives upstream at `iaa/users-roles` and is not yet represented as a dedicated catalogue folder.
