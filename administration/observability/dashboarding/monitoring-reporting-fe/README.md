# monitoring-reporting-fe

Frontend application for monitoring **reports and dashboards** — visualises the data collected by [eck-monitoring](../../performance-monitoring/eck-monitoring/README.md) and surfaces operational reports for Simpl agents and the broader infrastructure.

The production code lives at the canonical location — see [CANONICAL.md](CANONICAL.md). Machine-readable form: [`.canonical.yaml`](.canonical.yaml).

## Status

**UI scaffolding** — the upstream repo has an initial commit (≈ April 2026) that lays down the project skeleton; no business logic or dashboards yet. Catalogued here so the slot exists in the repo structure as the UI matures.
