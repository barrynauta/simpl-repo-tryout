# IAA UI Tests

Browser-driven end-to-end tests for the IAA frontend applications (identity provider UI, security attributes UI, users-and-roles UI, authentication provider UI, onboarding UI). Complements the API-level suite at [`iaa-api-tests/`](../iaa-api-tests/README.md).

Provenance: built by Simpl. Source repository: `iaa/test-automation-ui`. Owner: IAA team. Licence: EUPL 1.2.
