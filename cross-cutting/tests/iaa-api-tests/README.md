# IAA API Tests

API-level automated tests for the IAA (Identity, Authentication & Authorisation) services on a deployed Simpl-Open agent. Drives the Tier 1 / Tier 2 authentication, identity, and authorisation endpoints to validate behaviour across the full IAA stack.

Provenance: built by Simpl. Source repository: `iaa/test-automation`. Owner: IAA team. Licence: EUPL 1.2.
