# IAA CLI

> 🚧 **Repository archived upstream.** Kept here as a configuration *example* only. Do **not** install components from this repository — the production Helm charts live alongside each component.

Originally a Kubernetes-job-driven CLI for IAA component initialisation (authority and participant agent bootstrap, Keycloak client configuration). Useful as a reference when wiring authority/participant initialisation flows.

Provenance: archived upstream — no longer maintained. Source repository: `iaa/cli`. Owner: IAA team (historical). Licence: EUPL 1.2.
