# Installation Guide

To use this library or integrate the Spring Boot Starter, add the following dependency to your `pom.xml`:

```xml
<dependency>
    <groupId>eu.europa.ec.simpl</groupId>
    <artifactId>simpl-spring-boot-starter</artifactId>
    <version>2.6.0-RC</version>
</dependency>
```

Once added, the library will be available in your Spring Boot project.
