# Participant Common

Shared Java utilities for participant-side microservices (Provider Agent and Consumer Agent). Cross-cutting concerns reused across SD-Tooling, Contract Consumption, Catalogue Client, and the EDC Connector Adapter.

Provenance: built by Simpl. Source repository: `data1/common`. Owner: Data1 team. Licence: EUPL 1.2.
