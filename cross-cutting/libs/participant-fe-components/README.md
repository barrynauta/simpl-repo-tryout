# Participant FE Components

Vue.js implementation of the **SIMPL Open component library** — colour palette (Tailwind config), typography, and reusable UI components aligned to the [Simpl Open Figma design system](https://www.figma.com/design/UKm3Nzm9scM0Lvi72BjeXg/Simpl-Open).

Built on **PrimeVue (unstyled mode)** with **TailwindCSS** for styling.

Provenance: built by Simpl. Source repository: `data1/simpl-vue-components`. Owner: Data1 team. Licence: EUPL 1.2.
