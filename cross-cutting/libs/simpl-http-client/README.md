# Simpl HTTP Client

Java utility module for building **secure, configurable HTTP/2 clients** with custom SSL/TLS settings, certificate-revocation validation (CRL/OCSP), and optional ephemeral-proof preflight support. Used by Simpl-Open services that need to reach Tier 1 / Tier 2 endpoints with strong client-cert authentication.

Provenance: built by Simpl. Source repository: `iaa/simpl-http-client`. Owner: IAA team. Licence: EUPL 1.2.
