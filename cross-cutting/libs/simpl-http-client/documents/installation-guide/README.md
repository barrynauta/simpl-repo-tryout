# Installation Guide

To include this library in your project, please refer to the [Installation section](https://code.europa.eu/simpl/simpl-open/development/iaa/simpl-http-client/-/tree/main/simpl-http-client-okhttp#installation) of the simpl-http-client-okhttp module.  

You can also check the [Feign module](https://code.europa.eu/simpl/simpl-open/development/iaa/simpl-http-client/-/tree/main/simpl-http-client-feign) for additional configuration options and integration examples.
