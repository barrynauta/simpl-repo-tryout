# Participant Common Adapter

Adapter helpers used by participant-side services to talk to external integrations (the EDC Connector, the Catalogue, and the IAA gateways). Sits one layer above [`participant-common`](../participant-common/README.md).

Provenance: built by Simpl. Source repository: `data1/common-adapter`. Owner: Data1 team. Licence: EUPL 1.2.
