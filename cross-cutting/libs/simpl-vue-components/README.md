# simpl-vue-components

Vue 3 implementation of the **SIMPL Open design system** — colour palette (Tailwind config), typography, icons, and reusable UI components aligned to the [Simpl Open Figma design system](https://www.figma.com/design/UKm3Nzm9scM0Lvi72BjeXg/Simpl-Open). Built on **PrimeVue (unstyled mode)** + **TailwindCSS v4**, distributed as the npm package `@simpl/vue-components`.

The production code lives at the canonical location — see [CANONICAL.md](CANONICAL.md). Machine-readable form: [`.canonical.yaml`](.canonical.yaml).

## Used by

Imported as an npm dependency (`@simpl/vue-components`) by every Simpl-Open Vue/Astro frontend, including:

- [simpl-sd-ui](../../../data/semantics-and-vocabulary/schema-management/simpl-sd-ui/README.md)
- [simpl-schema-manager-ui](../../../data/semantics-and-vocabulary/schema-management/simpl-schema-manager-ui/README.md)
- [contract-manager-ui](../../../governance/contract-management/contract-establishment/contract-manager-ui/README.md)
- [simpl-catalogue-client](../../../integration/resource-discovery/search-engine/simpl-catalogue-client/README.md)

A Storybook instance ships with the library for component preview during development (`npm run dev` → `:6006`).
