# Participant Common Tier2

Tier 2 specific shared utilities for participant-side services — extends [`participant-common`](../participant-common/README.md) with helpers needed to interact with the Tier 2 authentication path (mTLS handshake, security-attribute resolution, ABAC enforcement glue).

Provenance: built by Simpl. Source repository: `data1/common-tier2`. Owner: Data1 team. Licence: EUPL 1.2.
