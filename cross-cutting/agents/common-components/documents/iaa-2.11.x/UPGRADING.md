# Upgrading from version `2.10.x` to version `2.11.x`

This file contains all you need to know to upgrade IAA components from version `2.10.x` to version `2.11.x`

Update your helm repository before start.

```shell
helm repo update
```

- [Added](#added)
- [Changed](#changed)
- [Removed](#removed)

> **⚠️** Make sure to verify that each component chart is updated to the latest version of chart `2.10.x` before starting the upgrade process.
