# eIDAS Demo Node Deploy

Ansible playbook that deploys an **eIDAS Demo Node** with Apache Tomcat and Oracle JDK on Ubuntu. Automates the official [eIDAS-Node v2.9 Installation Guide](https://ec.europa.eu/digital-building-blocks/sites/spaces/DIGITAL/pages/881983624/eIDAS-Node+version+2.9) so a working demo node can be brought up quickly for integration testing.

> ⚠️ **Demo / development purposes only.** Not intended for production. For production deployments follow the official eIDAS-Node documentation directly.

Provenance: built by Simpl. Source repository: `iaa/eidas-demo-node-deploy`. Owner: IAA team. Licence: EUPL 1.2.

## Pairs with

- [eidas-demo-keycloak-extension](../eidas-demo-keycloak-extension/README.md) — Keycloak SPI that integrates with the demo node deployed by this playbook.
