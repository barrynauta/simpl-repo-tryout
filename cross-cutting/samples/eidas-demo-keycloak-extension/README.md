# eIDAS Demo Keycloak Extension

Custom **Keycloak SPI** that lets a Keycloak realm authenticate end-users via an **eIDAS Demo Node** as an external identity provider. Used to verify the Tier 1 path against an eIDAS-compatible IdP without depending on a national production node.

> Demo / development purposes only.

Provenance: built by Simpl. Source repository: `iaa/eidas-demo-keycloak-extension`. Owner: IAA team. Licence: EUPL 1.2.

## Pairs with

- [eidas-demo-node-deploy](../eidas-demo-node-deploy/README.md) — the Ansible playbook that stands up the demo eIDAS Node this extension talks to.
