# Echo Frontend

Sample frontend that pairs with [`echo-backend`](../echo-backend/README.md). Demonstrates how a Simpl-Open frontend integrates with the Tier 1 / Tier 2 IAA stack and the standard authentication flow.

Used as a starting template by teams building new frontends.

Provenance: built by Simpl. Source repository: `iaa/echo-frontend`. Owner: IAA team. Licence: EUPL 1.2.
