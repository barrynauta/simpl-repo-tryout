# Angular Shell (eUI)

Host **shell** for the Simpl-Open microfrontend framework. Loads framework-specific remote applications at runtime via Webpack Module Federation. See `INTRODUCTION.md` and `GUIDELINES.md` in the source repo for the architecture overview and coding conventions.

Provenance: built by Simpl. Source repository: `iaa/microfrontend-framework/angular-shell`. Owner: IAA team. Licence: EUPL 1.2.
