# eDelivery

Sample / placeholder for the eDelivery integration with Simpl-Open. eDelivery is the EU CEF Building Block for secure, reliable, and traceable file/message exchange. The Simpl Connector exposes a hook that triggers an eDelivery transfer; the eDelivery service that consumes the hook lives at this upstream repository (currently a stub).

The production code lives at the canonical location — see [CANONICAL.md](CANONICAL.md). Machine-readable form: [`.canonical.yaml`](.canonical.yaml).
