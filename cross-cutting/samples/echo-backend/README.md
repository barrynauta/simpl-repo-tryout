# Echo Backend

Sample Java service demonstrating the **Tier 1 / Tier 2 IAA integration pattern** that any Simpl-Open backend microservice should follow. Includes the keystore configuration, Tier 1 client-cert handshake, Tier 2 token validation, and the recommended HTTP-client setup.

Used as a starting template by teams building new backend services.

Provenance: built by Simpl. Source repository: `iaa/echo-backend`. Owner: IAA team. Licence: EUPL 1.2.
